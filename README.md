# 🚀 Portfólio Ives Mendes

Um portfólio moderno e sofisticado desenvolvido com HTML, CSS e JavaScript puro, projetado para impressionar recrutadores e mostrar suas habilidades técnicas e profissionais.

## ✨ Características

- **Design Moderno**: Interface elegante com gradientes, sombras e animações suaves
- **Totalmente Responsivo**: Funciona perfeitamente em todos os dispositivos
- **Performance Otimizada**: Carregamento rápido e animações fluidas
- **Acessibilidade**: Navegação por teclado e foco visual adequado
- **SEO Otimizado**: Meta tags e estrutura semântica

## 🎯 Seções do Portfólio

### 1. **Hero Section**
- Apresentação impactante com gradiente de fundo
- Cards flutuantes animados
- Botões de call-to-action
- Indicador de scroll animado

### 2. **Sobre Mim**
- Descrição profissional
- Estatísticas animadas (experiência, projetos, comprometimento)
- Layout em grid responsivo

### 3. **Habilidades**
- **Soft Skills**: Trabalho em equipe, resolução de problemas, gestão de tempo, etc.
- **Hard Skills**: HTML5, CSS3, JavaScript, React, Node.js, Python, etc.
- Barras de progresso animadas
- Cards interativos com hover effects

### 4. **Projetos**
- Grid responsivo de projetos
- Overlay com links para GitHub e demo
- Tags de tecnologias utilizadas
- Efeitos de parallax sutil

### 5. **Experiência Profissional**
- Timeline vertical elegante
- Marcadores visuais
- Lista de conquistas por posição

### 6. **Contato**
- Informações de contato com ícones
- Formulário funcional com validação
- Sistema de notificações
- Links para redes sociais

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: 
  - Grid e Flexbox para layouts
  - CSS Variables (Custom Properties)
  - Gradientes e sombras
  - Animações e transições
  - Media queries para responsividade
- **JavaScript ES6+**:
  - Intersection Observer API
  - Event listeners
  - DOM manipulation
  - Local Storage
  - Performance optimizations

## 🚀 Como Usar

### 1. **Clone ou Download**
```bash
git clone https://github.com/seu-usuario/portfolio-ivesmendes.git
cd portfolio-ivesmendes
```

### 2. **Personalização**
- Edite o arquivo `index.html` para alterar conteúdo
- Modifique `styles.css` para personalizar cores e estilos
- Ajuste `script.js` para funcionalidades específicas

### 3. **Executar**
- Abra `index.html` em qualquer navegador moderno
- Ou use um servidor local:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx serve .

# PHP
php -S localhost:8000
```

## 🎨 Personalização

### Cores
As cores principais estão definidas como variáveis CSS no arquivo `styles.css`:

```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #f59e0b;
    --accent-color: #10b981;
    /* ... outras cores */
}
```

### Fontes
O portfólio usa a fonte Inter do Google Fonts. Para alterar:

```css
body {
    font-family: 'Sua Fonte', -apple-system, BlinkMacSystemFont, sans-serif;
}
```
